## Scripts

```bash
# Lint with `oxc`
yarn lint

# Run typescript based demo code with `swc`
yarn demo

# Run typescript based unit tests with `swc` + `jest`
yarn test

# Build with `esbuild`
yarn build

# Publish to npm
yarn release
```
